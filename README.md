# QA Automation Test Project

Automated test suite for validating web application functionality using Selenium WebDriver and TestNG.

## Test Cases

**Test 1 - Homepage Validation**
- Navigates to homepage
- Verifies page loads correctly
- Checks main UI components

**Test 2 - Job Filtering**
- Opens Quality Assurance careers page
- Applies location and department filters
- Validates filtered results appear

**Test 3 - Filter Verification**
- Confirms all jobs match selected filters
- Checks department and location fields

**Test 4 - Application Redirect**
- Clicks on job posting
- Verifies redirect to application page

## Tech Stack

- Java 11
- Selenium WebDriver 4.16.1
- TestNG 7.8.0
- Maven

## Setup

Requires Java 11+ and Chrome browser.
